Мета проєкту

Метою даного проєкту є створення консольного чат-бота мовою **Python**, який допомагає користувачу вести облік особистих фінансів. Програма дозволяє встановлювати бюджет, додавати витрати, переглядати їх за різними параметрами та отримувати фінансову статистику.

Основні можливості програми:

* встановлення загального бюджету;
* додавання нових витрат;
* перегляд списку всіх витрат;
* пошук витрат за датою;
* перегляд витрат за певний період;
* фільтрація витрат за категоріями;
* підрахунок залишку бюджету;
* формування звіту по категоріях.

Проєкт демонструє роботу з:

* функціями Python;
* файлами;
* форматом **JSON**;
* обробкою введення користувача;
* базовою логікою чат-бота.


Логіка та алгоритм роботи бота

Програма працює у вигляді консольного чат-бота, який взаємодіє з користувачем через введення команд.

Алгоритм роботи програми:

1. При запуску програма перевіряє наявність файлу finance_data.json.
2. Якщо файл існує — дані завантажуються з нього.
3. Якщо файл відсутній — створюється нова структура даних (бюджет і список витрат).
4. Після запуску бот вітає користувача.
5. Користувач вводить команду.
6. Програма аналізує введену команду та викликає відповідну функцію.
7. Функція виконує необхідну дію:

   * додає витрату;
   * показує витрати;
   * формує звіт;
   * обчислює залишок бюджету.
8. Після кожної зміни дані зберігаються у JSON-файл.
9. Програма працює у циклі while, поки користувач не введе команду "вийти".

Таким чином реалізовано просту систему керування фінансами через текстові команди.

---

Структура даних

Для збереження інформації використовується файл **finance_data.json**.

У ньому зберігається структура:

```json
{
    "budget": 5000,
    "expenses": [
        {
            "amount": 150,
            "category": "їжа",
            "date": "2026-03-10",
            "comment": "обід"
        }
    ]
}
```

### Пояснення структури:

budget — загальний бюджет користувача.

expenses — список витрат.

Кожна витрата містить:

* amount — сума витрати
* category — категорія витрати (їжа, транспорт, розваги тощо)
* date — дата витрати
* comment — додатковий коментар

Це дозволяє легко зберігати та обробляти фінансову інформацію.

Переконатися, що на комп’ютері встановлено **Python 3**.

Перевірити можна командою:

```
python --version
```

---

### 2. Завантажити проєкт

Скопіювати файли з GitHub або завантажити репозиторій.

У папці проєкту має бути файл:

```
bot.py
```

---

3. Запустити програму

У терміналі або командному рядку виконати:

```

```

---

4. Використовувати команди

Після запуску можна вводити команди:

* допомога
* встановити бюджет
* додати витрату
* показати витрати
* витрати за дату
* витрати за період
* витрати за категорією
* залишок
* звіт за категоріями
* вийти
